# Document Merger

A Python project for merging multiple documents into one.

## Description
This project provides a simple and efficient way to combine several documents into a single file. It supports various document formats and is ideal for tasks that require document consolidation.

## Features
- Merge multiple document formats (PDF, DOCX, TXT)
- Command-line interface for easy usage
- Customizable options for file handling

## Installation
To install the required dependencies, you can use pip:
```bash
pip install -r requirements.txt
```

## Usage Examples
You can merge documents using the command line:
```bash
python merge_documents.py file1.pdf file2.docx -o merged_document.pdf
```

## Requirements
- Python 3.6+
- pip

Make sure to have the necessary libraries installed as per the `requirements.txt` file.
